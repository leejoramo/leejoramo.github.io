Web Presence Video

I chatted with Web Presence support about not being able to add MP4 HTML 5 <video> to pages. The GUI based editor only supports using Flash video for uploaded video and the insertion of external video such as YouTube and Vimeo.

We can upload MP4 videos to the Web Presence Media Library, but it is placed in the page via Flash embed tags. We can get it to work by manually inserting the following HTML

    <video controls="" width="800">
    <source src="/UserFiles/Servers/Server_5224225/Media/VideoFileName.mp4">
    Your browser does not support the video tag.
    </video>

Evaluated Screencast software

I have tried two screencasting software and found them lacking:

  • Microsoft’s Windows 10 has a built in screen cast feature. This is mostly for integration wtih X-Box, and does not provide many tools beyond just grabbing the screen. (No mouse clicks, Keyboard strokes, editing, resizing, etc.). Created the “Vacation Request” screencast with this.

  • ActivePressenter: this is geared to the creation of interactive online classes. For this main task it looks very nice, but it is rather difficult to use for simplier screen captures. Created the Incident Management with this.

I have used two macOS apps extensively, ScreenFlow and IShowU both of which are very nice at creating screencasts. Still looking for something similar on windows.
